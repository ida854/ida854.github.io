Tre olika sätt att publicera material på webben

Här kollar vi lite på tre sätt att publicera material

  1. Google Sites
  2. Github Pages
  3. Google Cloud

Google Site och Github Pages är free hosting vilket betyder att man kan publicera sin webbplats på nätet utan kostnad, däremot dett finns andra spärrar, exempelvis liten lagringsstorlek, ingen kontroll över server/programvara, ingen databas, låst till ett visst domännamn.

Å andra sidan tittar vi lite påGoogle cloud som är VPS(Virtual private server), En VPS kan ses som ett mellanting mellan en dedikerad server och delad hosting (webbhotell). Oftast är det det ganska dyrt att flytta direkt till en egen dedikerad server och då finns möjligheten till att köra på en VPS istället, detta ger dig betydligt mer resurser än vad en delad miljö skulle kunna erbjuda. I stora drag kan man säga oavsett priset Google cloud som är VPS är bättre än de två free hosting eftersåm det gäller prestanda, stabilitet, skalbarhet, säkerhetskopiering och säkerhet.

Lyckligtvis för att komma igång med dem finns många vägledning och hjälpstart på nätet där man kan se likheter och skillnader mellan dem och gemföra de.För att ha ett bättre bild av deras funktioner och visa hur man kan publicera webbplats med hjälp av de tre och samt gemföra dem,använder jag av filmer som finns på youtube och visar klart, tydligt och stäg för stäg hur man kommer igång med dem.

Google Sites

Några Egenskaper

  • Google Sites är gratis att använda och det kostar inget att ha webbplatsen upplagd på Internet
  • Google Sites har svenska texter - men de omfattande hjälptexterna finns ännu inte på svenska. I värsta fall kan du översätta dem med Google Translate eller med Nice Translator.
  • Google Sites är byggt för samarbete - du kan bjuda in personer som kan skriva, kommentera, lägga in bilder och videofilmer och du kan välja vad du vill publicera
  • Det är lätt att komma igång med programmet, som erbjuder många finesser utan att kräva IT-kunskaper. Mycket är förberett men utseendet kan anpassas efter dina behov.

Github Pages

Några Egenskaper

  • Github sidor tillåter utvecklare att vara värd för en statisk webbplats för sina projekt för gratis.
  • Det är en av github viktigaste anläggningar som har gjort att upprätthålla fria open source-projekt mer ekonomisk speciellt för indieutvecklare.
  • Skapa en Github sida är enkel och snabb.
  • GitHub är en tjänst som ger oss utvecklare möjlighet att, via webben, skapa, hantera och lagra projekt och dess data via Git.
  • GitHub har också ett system för att skriva kommentarer, ToDo-listor m.m som underlättar vid arbetsgången.
