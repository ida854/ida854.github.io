Texteditor och tillägg

Texteditorn Visual Studio Code


Jag använder Visual Studio Code som är en gratis verktyg
för utvecklare och funkar bra.
Den är kraftfull, fungerar med många olika programmeringsspråk
och tekniker och är heller inte allt för krånglig att lära sig.


Tillägg

Det finns många användbara verkytg och tillägg som förenklar att skriva code i Visual Studio code arbetsmiljön. Bland de intrssanta tilläggen kan man påpeka på Linter(berör format och riktlinjer) Syntax Highlighter(För att tydligare kunna läsa koden skrivs den i olika färger), Code completion, Prettier(som i princip bara sköter om indentering (vänster indrag i koden). Detta gör att du tydligare ser var olika kodblock börjar och slutar. ) m.m som hjälper oss till att skriva koder lättare och på ett bättre sätt.
Jag tycker att mellan de tilläggen som Jag har installerat och provat prettier och Syntax Highlighter trots att de påverkar bara på utseende av koder är väldigt nyttiga och vägläder oss på ett bra sätt genom att ordna och markera våra koder. Jag är mycket nöjd med dem. 😊
